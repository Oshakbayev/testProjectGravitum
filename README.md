# User Management API

## Prerequisites

- Docker
- Docker Compose

## Setup

1. Install Docker
    - Download from [docker.com](https://docs.docker.com/get-docker/)

## Running

1. Start the application:
```bash
docker-compose up -d
```
Services will be available at:

API: http://localhost:8080
Swagger: http://localhost:8080/swagger/index.html#/
